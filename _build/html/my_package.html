5. 패키지 소스 설명¶

내 소스 설명하기¶

Title Name¶

My Module¶

class my_package.my_module.Test(a, b)[소스]¶

두 개의 int 값을 입력받아 다양한 연산을 할 수 있도록 하는 클래스.

매개변수

  • a (int) – a 값

  • b (int) – b 값

is_same()[소스]¶

미리 입력받은 a와 b값이 같은지 확인하여 결과를 반환합니다.

반환값

boolean True or False에 대한 결과, a와 b가 값으면 True, 다르면 False

예제:

다음과 같이 사용하세요:

>>> Test(1, 2).is_same()
False
plus()[소스]¶

미리 입력받은 a와 b값을 더한 결과를 반환합니다.

반환값

int a + b에 대한 결과

예제:

다음과 같이 사용하세요:

>>> Test(2, 1).plus()
3
